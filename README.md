## usage

compile the .cpp file and run the executable file.
the program will prompt you with various questions for customising the password.
simply input an int for the password length, and either 'y' or 'n' for the remaining questions.

## compiling the code:

to compile the code, run in local terminal:

```
$ g++ -p <desired_file_name> main.cpp
$ ./<desired_file_name>
```
